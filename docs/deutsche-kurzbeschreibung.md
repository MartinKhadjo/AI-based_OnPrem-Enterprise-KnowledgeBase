# Deutsche Kurzbeschreibung

Dieses Repository ist eine öffentliche, codefreie Showcase-Dokumentation für eine proprietäre, lokal betreibbare RAG-basierte Enterprise Knowledge Base.

Das System verarbeitet interne Rohdokumente zu durchsuchbaren Wissensartefakten. Dokumente werden registriert, optional gestaged, per OCR verarbeitet, partitioniert, gechunked, eingebettet und anschließend lexikalisch sowie vektoriell indexiert. Nutzer arbeiten über eine Browseroberfläche; die eigentliche Kontrolle liegt beim FastAPI-Backend. Antworten werden nicht direkt aus Dateien erzeugt, sondern aus verarbeiteten Artefakten und einem lokalen Modellserver.

## Professioneller Zweck

Das Repository zeigt Arbeitgebern und Headhuntern:

- Architekturverständnis;
- RAG-Engineering mit OCR, Chunking, Hybrid Retrieval und Reranking;
- saubere Trennung von Raw-Daten, Processed-Data und Anwendung;
- On-Premises-Denken für Datenschutz und Betrieb;
- Wartbarkeit, Debugging-Struktur und Betriebskonzepte;
- professionelle Dokumentation ohne Preisgabe des produktiven Codes.

## Nicht enthalten

Nicht enthalten sind Quellcode, Prompts, API-Keys, Modelle, Installer, interne Dokumente, Rohdaten, Vektorindizes oder Chat-Exporte. Für tiefergehende Einsicht sollte ein NDA, eine proprietäre Lizenz oder ein kontrollierter Code-Walkthrough verwendet werden.
